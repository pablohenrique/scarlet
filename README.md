scarlet
=======

A server in ruby.

## How to use
```sh
  $ ruby server.rb {[[conf.json],[folder_with_html,port]]}
```
Example:
```sh
  $ ruby server.rb conf.json
```
Or
```sh
  $ ruby server.rb "./public" 8888
```
